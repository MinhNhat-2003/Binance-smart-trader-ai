## 🤖 Binance Smart Trader AI

An AI-powered assistant that helps users analyze crypto assets and improve trading decisions.

## 🔎 Features
- BTC & ETH analysis  
- Portfolio suggestions  
- Trading strategies  
- Risk management insights  

## 🧠 AI Concept
This project simulates an AI trading assistant inspired by OpenClaw agent architecture, designed to provide intelligent responses based on user queries.
